OUT N ABOUT

← Volver al portfolio

Out N About es una aplicación móvil diseñada para facilitar la búsqueda, reserva y recomendación de eventos personalizados. El objetivo de este proyecto fue crear una experiencia digital útil, atractiva y socialmente interactiva para personas que quieren descubrir actividades que encajen con sus intereses.

El proceso comenzó con una fase de investigación centrada en entrevistas y encuestas, que nos permitió definir perfiles de usuario y comprender mejor sus hábitos. A partir de ahí, construimos mapas de empatía, recorridos de usuario y analizamos aplicaciones similares para detectar oportunidades de mejora.

Nuestra propuesta de valor se basa en cuatro pilares: recomendación personalizada según preferencias, sistema de reservas integrado, almacenamiento de actividades y una red social interna para compartir experiencias. Todo ello pensado para ofrecer una experiencia completa y centrada en el usuario.

En cuanto al diseño, desarrollamos wireframes, moodboards y un prototipo de alta fidelidad utilizando Figma. Optamos por una interfaz clara, moderna y dinámica, con un sistema visual coherente basado en el azul eléctrico como color principal. Este color transmite energía y confianza, elementos clave en una app que invita a salir, descubrir y conectar.

Out N About no solo permite buscar eventos, sino también vivirlos en comunidad. Los usuarios pueden compartir fotos, valorar experiencias y recomendar actividades a otros, generando una red social de ocio basada en gustos reales.

Este proyecto ha sido una oportunidad para aplicar fundamentos de UX/UI, diseño visual y pensamiento estratégico, siempre desde la empatía con el usuario. Representa una combinación de funcionalidad y diseño, y una muestra de cómo la tecnología puede potenciar nuestras experiencias cotidianas de forma sencilla y significativa.
